このドキュメントはサンプルのナレッジベースです。
以下のコンテンツに基づいて回答してください。

- プロジェクト名: Mini LLM Chat
- 用途: シンプルなチャット UI とバックエンド API の動作確認
- 主な機能:
  - OpenAI の Chat Completions API を利用した応答生成
  - システムプロンプトの指定と履歴を含む会話
  - API キー未設定時はスタブ応答
- 運用メモ:
  - バックエンドはポート 3001、フロントエンドはポート 5173 で起動
  - `/api/chat` に POST で messages/systemPrompt を送ると応答が返る
  - `/health` エンドポイントで状態確認が可能
